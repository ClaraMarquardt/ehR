gghelp {plotflow}R Documentation

Open Help Pages for ggplot2

Description

gghelp - Open Hadely Wickham's ggplot2 web page.

ggcook - Open Winston Chang's ggplot2 Cookbook for R page.

Usage

gghelp(FUN)

ggcook()

Arguments

FUN

A particular ggplot function to reference. Default is the index page.

Value

Opens a help web page.

browseURL

Examples

## Not run: 
gghelp() 
gghelp("theme")
ggcook()

## End(Not run)
