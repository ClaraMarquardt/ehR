qsymbol {plotflow}R Documentation

Plot ggplot2/base R Symbols

Description

Makes a plot of ggplto2/base R symbols.

Usage

qsymbol(ggplot2 = TRUE)

Arguments

ggplot2

logical. If TRUE ggplot2 symbols will be plotted. If FALSE base graphics symbols will be plotted.

Examples
