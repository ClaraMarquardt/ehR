unbalanced_facet_axis {plotflow}R Documentation

Add Tick Marks to an Unbalanced facet_wrap

Description

Adds the tick marks to an unblanced grouping of facet_wrap plots.

Usage

unbalanced_facet_axis(ggplot_obj, position = c("up", "down"))

Arguments

ggplot_obj

An unbalanced ggplot2 facet_wrap object.

position

Either "up" (match unbalanced facet's postition) or "down" (along bottom most axis).

Author(s)

Julius (stackoverflow.com)

References

http://stackoverflow.com/a/13316126/1000343

Examples

set.seed(10)
mtcars[["new"]] <- sample(LETTERS[1:7], nrow(mtcars), TRUE)

library(ggplot2)

unbalanced_facet_axis(ggplot(mtcars, aes(x=mpg, y=hp)) +
    geom_line() +
    facet_wrap(~new, ncol=2))

unbalanced_facet_axis(ggplot(mtcars, aes(x=mpg, y=hp)) +
    geom_line() +
    facet_wrap(~new, ncol=3), "down")
